# Barber Booking System 

A web app for managing barber shop appointments.  
Built with **HTML, CSS, JS, PHP, and MySQL**.

## Features
- Customers can view time slots, book, reschedule, or cancel.
- Admin dashboard for managing bookings.
- Email/SMS confirmations.
- Export bookings to CSV.

## Setup
1. Clone the repo
2. Import `sql/barber_booking.sql` into MySQL
3. Run with XAMPP (htdocs/barber-booking)
